## Quick Reference

### Key Technologies

- **Frontend**: React 18 + TypeScript + Vite
- **Backend**: Express.js + TypeScript
- **Database**: PostgreSQL + Drizzle ORM
- **Storage**: Google Cloud Storage
- **Auth**: Replit Auth (OIDC)
- **UI**: shadcn/ui + Tailwind CSS

### Common Commands

```bash
# Development
npm run dev          # Start dev server (port 5000)
npm run check        # Type check
npm run build        # Build for production

# Database
npm run db:push      # Push schema changes to database
```

### Entry Points

- **Frontend**: `client/src/main.tsx` → `App.tsx`
- **Backend**: `server/index.ts`
- **Database Schema**: `shared/schema.ts`
- **API Routes**: `server/routes.ts`

## Glossary Preview

Common terms you'll encounter:

- **Object Storage**: Google Cloud Storage backend for file data
- **Share Link**: Token-based temporary file access URL
- **Presigned URL**: Time-limited upload/download URL generated by storage service
- **Drizzle ORM**: Type-safe database query builder
- **shadcn/ui**: Component library built on Radix UI primitives

See [90_GLOSSARY.md](./90_GLOSSARY.md) for complete definitions.
